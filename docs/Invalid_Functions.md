# GPU 互換性に関する注意 (Invalid Functions)

現在の Fuse GPU 環境（OpenCL/Metal等）でビルドエラーまたは予期せぬ動作を引き起こすことが確認された関数・マクロの一覧です。

| 関数・マクロ | 状況 | 理由 | 判明日 |
| :--- | :--- | :--- | :--- |
| `_length(v)` | NG | ビルドエラー（未定義のシンボル） | 2026-03-21 |
| `_dot3f(v1, v2)` | NG | ビルドエラー（未定義のシンボル） | 2026-03-21 |
| `_stepf(edge, x)` | NG | ビルドエラー（未定義のシンボル） | 2026-03-21 |
| `_fmod(x, y)` | **Behavior Differ** | 負の数に対する挙動が `floor` ベースと異なるため、模様がズレる | 2026-03-21 |
